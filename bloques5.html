Bloques De Código: Eventos

Eventos

Los bloques de eventos son la quinta categoría en la paleta de bloques y tienen un código de color amarillo.

Estos bloques son responsables de iniciar una pila o un programa cuando se produce un evento específico y de transmitir mensajes.

En la categoría de bloques de eventos hay un total de 12 bloques, 10 bloques de sombrero, 2 bloques de pila.

Cuando comienza el programa

Ejecuta todos los bloques adjuntos a él, secuencialmente de arriba a abajo, cuando se inicia el programa. El programa se puede iniciar utilizando el botón Descargar y ejecutar, o seleccionando y ejecutando el programa desde el Bloque EV3.

Cuando el color es

Ejecuta los bloques adjuntos a él cuando el sensor de color detecta un color específico. El sensor puede detectar:

- Negro
- Marrón
- Azul
- Celeste
- Verde
- Amarillo
- Rojo
- Blanco
- Sin color
- Cualquier cambio de color

Este bloqueo solo se activará en caso de detectar el color especificado. Esto significa que el bloque no se volverá a disparar mientras el color detectado permanezca sin cambios.

Cuando se presiona

Ejecuta los bloques adheridos a él cuando se presiona o suelta el sensor táctil.

Este bloque solo se activará en el evento especificado. Esto significa que el bloqueo no se volverá a disparar mientras el estado del sensor táctil permanezca sin cambios.

Cuando la distancia es

Ejecuta los bloques adheridos a él cuando la distancia del sensor ultrasónico a un objeto es menor, mayor, igual o cambia más que la distancia especificada en centímetros o pulgadas.

Este bloque solo se activará en el evento especificado. Esto significa que el bloqueo no se volverá a disparar si la distancia permanece sin cambios.

Cuando el ángulo es

Ejecuta los bloques adheridos a él cuando el ángulo del sensor giroscópico es menor, mayor, igual o cambiado más que el ángulo especificado.

Este bloque solo se activará en el evento especificado. Esto significa que el bloque no se volverá a disparar si el ángulo permanece sin cambios.

Cuando se presiona el botón de ladrillo

Ejecuta los bloques adjuntos a él cuando se presiona o suelta el botón de ladrillo especificado.

Este bloque solo se activará en el evento especificado. Esto significa que el bloque no se volverá a disparar mientras el Botón de Bloque permanezca en su estado actual.

Cuando

Ejecuta los bloques adjuntos a él cuando su condición booleana es verdadera.

Este bloqueo solo se activará en caso de que la condición se cumpla. Esto significa que el bloque no se volverá a disparar si la condición permanece verdadera.

Cuando recibo un mensaje

Ejecuta los bloques adjuntos cuando el mensaje especificado se transmite mediante el mensaje de transmisión o el mensaje de transmisión y el bloque de espera.

Mensaje de difusión

Transmite el mensaje especificado. Se activarán todos los bloques de Cuando reciba mensajes que se hayan configurado para el mensaje especificado.

Este bloque de transmisión envía el mensaje especificado e inmediatamente pasa al siguiente bloque, a diferencia del Mensaje de difusión y Bloque de espera, que esperará hasta que todas las pilas de programación "escuchando" este mensaje hayan terminado antes de proceder.

Mensaje de difusión y esperar

Transmite el mensaje especificado. Se activarán todos los bloques de Cuando reciba mensajes que se hayan configurado para el mensaje especificado.

Este bloque de transmisión envía el mensaje especificado y espera hasta que todos las pilas de programación "Escuchando" este mensaje ha terminado antes de pasar al siguiente bloque de su propia pila. Esto es diferente al mensaje de difusión sin espera, que procederá sin esperar.

Cuando el temporizador

Ejecuta los bloques adjuntos a él cuando el temporizador excede el valor especificado.

Este bloqueo solo se activará en caso de temporizador exceder el valor especificado. Esto significa que el bloque no se volverá a disparar mientras el temporizador esté por encima del valor especificado.
