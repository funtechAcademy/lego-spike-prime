Bloques De Código: Variables

Variables

Los bloques variables son la tercera categoría en la paleta de bloques y tienen un código de color naranja oscuro.

Al iniciar un nuevo proyecto no habrá ningún bloque en esta categorí a, pero puedes usar las herramientas de este bloque para crear variables (almacenar valores) y matrices (listas de almacenamiento).

Definir variable

Antes de que aparezcan los bloques de las Variables, tenemos que crear una de ellas. Pulsaremos en Crear una variable para crearla y nos aparecerá la siguiente pantalla donde le daremos el nombre:

A partir de tener la primera variable, nos aparecerán los bloque relativos a ellas.

Variable

Devuelve el valor de una variable. Siempre que se crea una variable, aparece una versión de este bloque con el nombre de pila de la variable, lo que da como resultado una versión de este bloque para cada variable.

Cada versión del bloque tiene su valor "asignado".

Establecer variable en

Establece la variable en el valor especificado. La variable puede ser una cadena o un número.

Cambiar variable por

Cambia la variable por un valor especificado. Si la variable es una cadena de texto y no un nómero, se establece en el valor por el que se iba a cambiar la variable (convirtiendo la cadena en un "0").

Definir Listas

Para las Listas, nos ocurre lo mismo que para las Variables, antes de que aparezcan los bloques de las Listas, tenemos que crear una de ellas. Pulsaremos en Crear una lista para crearla y nos aparecerá la siguiente pantalla donde le daremos el nombre:

Lista

Devuelve el valor de una lista. Siempre que se crea una lista, aparece una versión de este bloque con el nombre de pila de la lista, lo que da como resultado una versión de este bloque para cada lista.

Agregar artículo a la lista

Agrega un elemento que contiene el texto o el número especificado a una lista.

Eliminar todos los elementos de la lista

El bloque elimina todos los elementos de la lista especificada.

Reemplazar el elemento del índice de la lista con otro elemento

Reemplaza el contenido del elemento especificado por un nuevo valor.

Valor del artículo en el índice en lista

Devuelve el valor del elemento especificado en una lista.

Longitud de la lista

Devuelve el número de elementos contenidos en la lista especificada.
